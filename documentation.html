Getting Started

  1. Download LUMA OS from the downloads section
  2. Extract the archive (tar.gz or zip)
  3. Run the installation script: sudo ./install.sh
  4. Start LUMA OS: lumaos activate
  5. Say "Computer" followed by your voice command
